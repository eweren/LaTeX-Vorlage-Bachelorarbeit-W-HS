\section{Entwicklung}\label{sec:development}
Wenn Code eingebettet werden soll, geht das mit der lstlisting Funktion:
\begin{lstlisting}
    // Netter Democode
    if (deadline === 'isNear') {
        this.code();
    } else if (deadline === 'tomorrow') {
        this.sweat();
    } else if (deadline === 'inSomeHours' {
        this.hustle();
    } else if (deadline === 'inThePast') {
        this.chill();
    }
\end{lstlisting}
